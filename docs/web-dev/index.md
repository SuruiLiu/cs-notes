# Web 开发

这个部分包含了 Web 开发相关的学习笔记，涵盖前端、后端以及全栈开发的各个方面。

- HTML/CSS 基础
- JavaScript 核心
- 现代前端框架
  - React
  - Vue
  - Angular

### 后端开发
- Node.js
- Python Web 框架
- 数据库
- API 设计

### 全栈开发
- MERN Stack
- JAMStack
- 微服务架构
